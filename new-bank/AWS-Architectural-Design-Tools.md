# AWS Architectural Design Tools

## 1. AWS Well-Architected Tool

- **Service:** AWS Well-Architected Tool
- **Console:** AWS Management Console → Services → Well-Architected Tool
- **URL:** https://console.aws.amazon.com/wellarchitected/
- **Purpose:** Review and improve your cloud architectures using AWS best practices.

## 2. AWS Architecture Center

- **Website:** https://aws.amazon.com/architecture/
- **Purpose:** Reference architectures, diagrams, and design patterns for AWS solutions.

## 3. AWS Solutions Architect Tools

- **AWS Solutions Constructs:** https://aws.amazon.com/solutions/constructs/
- **AWS Perspective (for visualizing workloads):** https://aws.amazon.com/solutions/implementations/aws-perspective/

You can access these via the AWS Console or the AWS website. The Well-Architected Tool is the most direct for architectural reviews. If you need guidance on a particular AWS architecture tool or workflow, let me know!
